# Personal-finance-Manager
Personal Finance Management System is a Python-based CLI application that helps users track and manage daily expenses. It uses object-oriented design, CSV file handling for data storage, input validation, and a menu-driven interface to generate category-wise and monthly expense reports.


A Python-based command-line application to help users track, manage, and analyze their daily expenses efficiently. The project demonstrates object-oriented programming, file handling using CSV, modular design, and basic data analysis.

Features
Add new expenses with validation
View all recorded expenses
Category-wise expense summary
Monthly expense report with total and average
CSV-based data persistence
Backup of expense data
Menu-driven command-line interface
Error handling for invalid inputs

Technologies Used
Python 3
CSV module
Jupyter Notebook 
Standard Python Libraries
